# Analisis-IAM

Project ini merupakan implementasi Identity and Access Management (IAM)
menggunakan Supabase pada aplikasi web sederhana.

## Fitur
- Login
- Logout
- Role Admin
- Role User
- Row Level Security (RLS)
- JWT Authentication

## Teknologi
- HTML
- CSS
- JavaScript
- Supabase
  
## Cara Menjalankan
1. Clone repository
2. Buka index.html
3. Hubungkan ke Supabase

## Deployment
https://iamdokumen.netlify.app

## Hasil Pengujian
- Login berhasil
- User hanya melihat datanya sendiri
- Admin dapat mengakses seluruh data
- RLS berhasil membatasi akses
